Авторизация на hh.ru:

API hh.ru требует OAuth-токен. Получите его:

  1. Зайдите на dev.hh.ru/admin
  2. Создайте приложение → получите Client ID и Client Secret
  3. Получите Access Token через OAuth или личный кабинет
  4. Вставьте токен в поле ниже

Без токена API может возвращать 403/400 — в таком случае используется fallback-парсинг HTML.
